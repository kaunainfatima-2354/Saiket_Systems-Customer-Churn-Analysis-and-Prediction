# Customer Churn Analysis and Prediction

This repository contains a **Machine Learning internship task** completed as part of the **Saiket Systems Internship Program**.  
The project focuses on analyzing customer data and building predictive models to identify customers likely to churn.

---

## Internship Details
- **Organization:** Saiket Systems  
- **Domain:** Machine Learning  
- **Project Type:** Internship Task Submission  

---

## Project Objective
To analyze customer behavior, identify key factors influencing churn, and develop machine learning models that can predict customer churn effectively.

---

## Work Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature encoding and scaling  
- Model training and comparison  
- Model evaluation using appropriate metrics  
- Interpretation of results and insights  

---

## Models Used
- Logistic Regression  
- Decision Tree Classifier  
- Random Forest Classifier  
- Gradient Boosting Classifier  

---

## Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook

- The project follows a standard machine learning pipeline.
